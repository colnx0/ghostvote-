# GhostVote 👻

GhostVote is a private voting app built on Solana.

## 🔗 Live App
https://ghost-vote-flow.base44.app

## 💡 What it does
- Create polls
- Vote anonymously
- See results without revealing identity

## 🔐 Privacy
Votes are hidden and only total results are shown.

## 🛠 Built with
- Base44
- Solana wallet
